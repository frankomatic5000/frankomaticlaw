# SKILL.md - powerpoint-creator

## Name
powerpoint-creator

## Description
Create professional PowerPoint presentations using python-pptx. Supports slides, charts, tables, images, and custom templates. FrankCMO can generate actual .pptx files from markdown content.

## Source
**ClawHub:** `claw-presentation-creator`
**Install:** `openclaw skills install claw-presentation-creator`

## Capabilities

- Create .pptx files from markdown/content
- Custom templates (Imigrou/GrowBiz branded)
- Charts and graphs from data
- Tables, images, bullet points
- Professional layouts

## Usage (Once Installed)

```
"FrankCMO: Create PowerPoint from imigrou_executive_presentation_PT.md"
"Generate slides with GrowBiz branding"
"Export marketing plan to PowerPoint"
```
